## Portfolio Projects

### [LLM Eval CI](https://github.com/alberezov-del/llm-eval-ci)

GitHub Action and CLI for regression testing LLM, RAG, and agent applications.
It runs JSONL eval suites, scores fact coverage, retrieval quality,
faithfulness, schema validity, latency, and cost, then blocks regressions with
baseline comparison and Markdown reports.

**Stack:** Python, Pydantic, PyYAML, jsonschema, pytest, GitHub Actions.

### [SupportAI Copilot](https://github.com/alberezov-del/support-ai-copilot)

End-to-end AI service for customer support automation. It classifies support
requests, retrieves grounded policy context, generates answers with sources,
routes tickets, stores feedback, and exposes API/UI metrics.

**Stack:** Python, machine learning, RAG, API, SQLite, Docker.
